なぜ Tealus？

既存メッセンジャーの課題を解決するために生まれました

💾

サーバー保存

画像・動画・ファイルはすべてサーバーに保存。端末の容量を圧迫しません。サムネイル自動生成で高速表示。

🔒

完全自社管理

オンプレミスで運用可能。データは社外に一切出ません。Docker Compose で簡単にデプロイ。

📱

PWA対応

スマホ・PCのブラウザからアプリとして利用可能。インストール不要、ホーム画面に追加するだけ。

🤖

AI連携

AIエージェントがチャットに参加。日常Q&Aから複雑なコード生成まで、3層構造で最適に処理。

📞

音声・ビデオ通話

WebRTC（mediasoup SFU）によるグループ通話。トランシーバー（PTT）モードも搭載。

🔊

TTS 読み上げ

AIの回答を音声で自動読み上げ。Aivis Cloud API による高品質日本語音声合成、10種類の声から選択。

LINEライクな
直感的チャット体験

使い慣れたメッセンジャーの操作感をそのままに、業務に必要な機能を詰め込みました。

  • リアルタイムメッセージ（Socket.IO）
  • 既読表示（未読数 + 既読数）
  • リプライ・メッセージ編集・削除
  • 絵文字リアクション（6種類）
  • メンション（@ユーザー名 / @all / @here）
  • メッセージ検索（全ルーム横断）
  • AIスタンプ自動生成（1パック16枚）
  • 音声メッセージ（Whisper文字起こし + AI整形）
  • TODO機能（タグベース）
  • カスタムテーマ・文字サイズ設定
