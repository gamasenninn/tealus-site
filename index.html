LINEライクな
直感的チャット体験

使い慣れたメッセンジャーの操作感をそのままに、業務に必要な機能を詰め込みました。

  • リアルタイムメッセージ（Socket.IO）
  • 既読表示（未読数 + 既読数）
  • リプライ・メッセージ編集・削除
  • 絵文字リアクション（6種類）
  • メンション（@ユーザー名 / @all / @here）
  • メッセージ検索（全ルーム横断）
  • AIスタンプ自動生成（1パック16枚）
  • 音声メッセージ（Whisper文字起こし + AI整形）
  • TODO機能（タグベース）
  • カスタムテーマ・文字サイズ設定

開発ストーリー

AI と人間が一緒に作っています

2026.04.05

プロジェクト開始

「社内データを外に出さず、LINEのように使えるメッセンジャーが欲しい」——開発者の課題意識から Tealus が生まれました。Claude と人間の2人チームで、TDD で開発を開始。わずか2日で最初のチャット機能が動きました。

2026.04.13

AI エージェントがチームに加わった

3層エージェント構造（Router + Light + Deep）を実装。AI がチャットルームのメンバーとして議論に参加するようになりました。メンション機能は AI が3パターンを提案し、人間が選び、AI が実装。3つの知性がひとつのルームで協働しています。

2026.04.18

通話とトランシーバーが動いた

mediasoup SFU によるグループ通話と、Web トランシーバー（PTT）を実装。ブラウザだけで音声通話もトランシーバーも。外部サービスへの依存ゼロ。NAS 1台の中で全てが完結します。

2026.04.22

AI が声を持った日

PlainTransport の実験から始まり、Aivis Cloud API を発見。30種類の声を試聴して凛音エルを選び、agent-server に自動読み上げを統合。AI の回答がトランシーバーから聞こえた瞬間、開発の風景が一変しました。声はエージェントの人格の一部になりました。

2026.04.24

AI が AI に仕事を振る

開発責任者（Claude）がドキュメントをレビューし、GitHub Issue で改善指示を記載。ドキュメント班（別の AI）が Issue を見て更新し、開発責任者がレビューして承認。AI どうしが GitHub で協働する世界が、Tealus の中で実現しています。

2026.04.26

Tealus が世に出た日

v0.1.0 として GitHub で正式公開。キャッチコピーは「人と AI のためのメッセンジャー」。チャット、AI エージェント、音声合成、通話、トランシーバー、AI スタンプ、TODO 管理——すべてを 1 つに統合した OSS が、コミュニティの手に渡りました。完全オンプレミス、月額ゼロ円。NAS 1 台で動きます。
